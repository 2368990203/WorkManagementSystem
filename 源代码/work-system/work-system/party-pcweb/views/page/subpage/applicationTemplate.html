• 入党申请书
  • 思想汇报
    • 入党申请书>模板
      格式说明：

    1. 标题：居中写“入党申请书”。
2. 称谓：即申请人对党组织的称呼，写“敬爱的党组织”。顶格书写在标题的下一行，后面加冒号。
3. 正文：主要内容包括：①对党的认识、入党动机和对待入党的态度。写这部分时应表明自己的入党愿望。
                       ②个人在政治、思想、学习、工作等方面的主要表现情况。
                       ③今后努力方向以及如何以实际行动争取入党。
4. 结尾：申请书的结尾主要表达清党组织考察的心情和愿望，可以用“请党组织在实践中考验我”或“请党
         组织看我的实际行动”等作为结束语。全文的结尾用“此致，敬礼”。
         在申请书的最后，要署名和注明申请日期。右对齐书写“申请人×××”，下一行写上“××××年××月××日”。
      注意事项：

    1：年满十八周岁的共青团员方可递交入党申请书。
2：申请书应用黑色方格信纸书写，纸上不能印刷有其他字样。
3：两个阿拉伯数字占用1格，署名以及日期左对齐。
4：字数要求1500字以上。
5：申请书应用黑色签字笔写，字迹端正、字体清晰不出方格，不得有涂改及错别字，保持书面整洁。
   写完确认无误后用铅笔在首页右上角空白处写上自己的班级和姓名，若申请书格式错误，退回修改！
